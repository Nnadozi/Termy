# Termy
Termy is a mobile app that helps you start your journey toward a smarter vocabulary, with just a few minutes a day.

## Built With
Frontend: React Native, Expo, TypeScript, and SQLite<br>
Backend: Supabase (PostgreSQL database + Edge Functions), Expo notifications
